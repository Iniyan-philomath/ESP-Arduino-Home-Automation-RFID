# Contributing

Thanks for contributing to this ESP Arduino project repository. Follow these simple guidelines to keep the repo clean and easy to use.

## Contribution process

1. Fork the repository or work in your own branch.
2. Make small, focused changes.
3. Test the sketch on the correct board in Arduino IDE or PlatformIO.
4. Use clear commit messages.
5. Create a pull request with a short summary of your changes.

## Code guidelines

- Keep Arduino sketches readable and well-commented.
- Use descriptive names for variables and functions.
- Avoid hardcoding sensitive values such as Wi-Fi credentials in shared code.
- Document any hardware changes or wiring updates clearly.

## Project structure

- `ESP8266-RFID-Room-Control/` contains the ESP8266 project.
- `ESP32-Drowsiness-Monitor/` contains the ESP32 project.
- Keep each project folder self-contained with its own README.

## Reporting issues

- Open a GitHub issue for bugs, feature requests, or questions.
- Provide a concise description of the problem and any error output.
- Include hardware details and board settings when possible.
